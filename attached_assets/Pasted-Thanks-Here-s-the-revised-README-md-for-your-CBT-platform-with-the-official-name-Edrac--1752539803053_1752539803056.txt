# 📚 Edrac – AI-Powered CBT & Learning Platform

**Edrac** is a state-of-the-art, AI-driven Computer-Based Testing (CBT) and smart learning platform built to serve **students, institutions, and corporate bodies** across **Nigeria and West Africa**. Edrac supports the **entire Nigerian curriculum**, major regional exam models (WAEC, NECO, JAMB, GCE, Post-UTME), and provides tools for **custom interview tests**, **corporate assessments**, and **AI-enhanced study support**.

---

## 🚀 Key Features

### 🤖 AI-Powered Learning

* **AI Question Generator** – Automatically creates questions for any topic or content.
* **24/7 AI Tutor** – On-demand tutoring and concept clarification.
* **Smart Feedback & Grading** – Instant results, detailed analytics, and tailored suggestions.

### 🧪 Full CBT Engine

* **Subject Bank** – Covers all subjects in Nigerian curriculum by class and difficulty.
* **Timed Exams** – Fully timed CBT with countdown, auto-submit, and answer randomization.
* **Anti-Cheating Measures** – Tab switch detection, focus monitoring, webcam lock-in (optional).
* **Exam Types Supported** – JAMB, WAEC, NECO, GCE, POST-UTME, in-house aptitude tests, interview simulations.

### 🗂 Custom Exam Builder

* For schools, HR teams, organizations, or trainers to create:

  * ✅ Interview tests
  * ✅ Certification tests
  * ✅ Corporate training assessments

### 🧑‍💼 Role-Based Dashboards

* **Admin** – Manage questions, users, payments, ads, and site content.
* **Student** – Take tests, review performance, ask AI, save progress.
* **Institutions** – Manage student pools, organize exams, monitor analytics.

### 💬 Learning Engagement Features

* **Ask AI Button** – Get AI explanations on any question during practice.
* **Comment Thread** – Discuss, ask for help, or answer peer questions.
* **View As** – Switch between exam formats for JAMB/WAEC/interview view.

## 🛠 Installation

```bash
# Clone the project
git clone https://github.com/yourusername/edrac.git
cd edrac

# Install dependencies
npm install

# Create environment variables
cp .env.example .env

# Fill in your env keys for database, JWT, payments, etc.

# Run in dev mode
npm run dev

# Build for production
npm run build
npm start
```
